キャラのステータス

下記が詳細です。

名前説明
レベル（Lvl）キャラの総合的な強さを示す値。自分よりレベルの高いペットは連れて歩けない。
性別　（SEX）男性か女性か。
生命力（HP）この値がゼロになるとペナルティを受けて、スタートの街に強制送還されます。
行動力（TP）３回戦闘をすると、１減少します。特技を使用する際もこの値を消費します。
この値がゼロのまま戦闘をするとHPが減少します。
ｷｬﾊﾟｼﾃｨ（CP）"生成屋でのみ使用できるお金"と考えてください。
レベルが上がった時のみ増加します。
所持金（ G ）お金です。
年齢　（AGE）キャラの年齢です。最初は１５歳から始まります。
攻撃力（DMG）値が大きい程、戦闘時に高いダメージを与えます。
防御力（A C）値が大きい程、戦闘時に受けるダメージを軽減します。
力　　（STR)戦闘時の敵に与えるダメージ値に影響
素早さ（AGI）戦闘時の敵の攻撃を避ける率、クリティカルヒット発生率に影響。
また、この値が150以上で戦闘時２回行動、300以上で３回行動が可能。
知性　（INT）属性攻撃のダメージ値、特技発動率に影響
魅力　（CHA）戦闘後のペット獲得率、戦闘中のペットの攻撃参加率に影響
火耐性（SVF）火属性攻撃に対する防御値
水耐性（SVC）水属性攻撃に対する防御値
魔耐性（SVM）魔属性攻撃に対する防御値
経験値（EXP）ゲージがいっぱいになるとレベルが一つ上がります。

　※ジョブによって多少効果が変わってきます。

ステータスバーの説明

　STR.AC.AGI.INT　にはバー（ゲージ）が存在します。
　紫色の部分がいっぱいになると、ステータスが１上昇します。

バーが上昇する条件は以下の通りです。
  • STR
    		• ・・・敵に攻撃を当てた回数。
  • AC
    		• ・・・敵から攻撃を受けた、総合ダメージ。
  • AGI
    		• ・・・敵の攻撃をよけた回数。
  • INT
    		• ・・・敵への攻撃をはずした回数。

    　わざと弱い武器や防具を装備したりするなどの工夫をする事で、
    　ステータスの成長が早まったりします。
