ペット

戦闘終了後、敵がペットになることがあります。
集めたペットは連れて歩くと戦闘に参加してくれます。
また、ペット合成で別のペットを生み出すことも可能です。
すでに獲得したペットはいなくなるまで再獲得はできません。

ペットの獲得確率はキャラのCHAステータスと連れ歩いているペット数が影響します。
特技で戦闘中にCHAの上がるジョブもありますがこれには影響されません。
ただし、ペット支援のCHA効果は効力が持続中に戦闘が終わると確率に影響されます。


ペット特技

次の特技のどれか一つをそれぞれ持っています。

  • 物理ダメージ
  • 火属性ダメージ
  • 水属性ダメージ
  • 魔属性ダメージ
  • 無属性ダメージ
  • HP回復
  • ACアップ
  • STRアップ
  • AGIアップ
  • INTアップ
  • CHAアップ

ここに載っていない特技もあります。
ペット成長

ペットはプレイヤーと同様、EXPを得てレベルアップします。
ペットのレベルが上がると、戦闘参加値(EF)と効果値(PP)がそれぞれ上がります。
※一度でも小屋に預けると、ペットのステータスは初期化されます。
