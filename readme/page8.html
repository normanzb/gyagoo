冒険者ギルド

  • 所持金を預ける
    		• ・・・所持金を預けられます。キャラが死んでも影響を受けません。
  • 預金を引き出す
    		• ・・・預金からお金を引き出します。
  • アイテムを預ける
    		• ・・・アイテムを保管出来ます。
  • アイテムを引き出す
    		• ・・・保管してあるアイテムを取り出します。
  • 福引券を預ける
    		• ・・・福引券を預けられます。
  • 福引券を引き出す
    		• ・・・福引券を引き出します。他のプレーヤーから送られてきた福引券は、ここに滞納されます。
  • アイテムを使う
    		• ・・・道具系アイテムを使用できます。
  • アイテムを装備する
    		• ・・・アイテムの装備はここでしかできません。また、各ジョブによって装備できる重量が違う点にも注意してください。 装備すると、以前装備されていたアイテムははずされます。
  • アイテムを捨てる
    		• ・・・いらないアイテムを投棄することができます。１度投棄したアイテムはもう戻らないことに注意。

    寺院

    ジョブの熟練度がMAXになると、他のジョブに転職する事ができます。
    極めたジョブの組み合わせによっては、転職できるジョブの数が増えていきます。
    ジョブによって、極めるまでに必要な熟練度は様々です。

    尚、極めたジョブは特技の発動確率が高くなります。

  • ボックスの中身を見る

    福引所

  • 福引をする
    		• ・・・福引券を１枚使用し、福引ができます。効果のいいアイテムほど当たる確立が低くなっています。　※景品一覧参照
  • 丸薬と交換
    		• ・・・福引券を１０枚使用し、丸薬を貰う事が出来ます。
  • 天龍丸薬と交換
    		• ・・・福引券を５０枚使用し、天龍丸薬を貰う事が出来ます。

    福引券は、戦闘終了後、稀に手に入ります。
    しかし、自分より弱い敵と戦っても、なかなか手に入りません。
    福引券を効率よく集めるには、自分のレベルに合った場所で戦闘をする事が重要です。
    福引券がなかなか手に入らなくなったら、もう少し強い敵と戦ってみましょう。
    イケニエ合体の館

    連れ歩いている２匹のペットを合体させます。
    ベースとなるペットを最初に選択し、次にイケニエとなるペットを選択します。

    ベースにしたペットが強化されるか弱化するかは、イケニエとして捧げたペットの能力によります。
