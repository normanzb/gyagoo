基本システム

　まず最初に「ゾーン」という言葉の説明をします。
ゾーンとはいわゆる「マップ」「フィールド」と同じ意味だと思ってください。
このゲーム内の世界はいくつものゾーンによって分けられています。

ゾーンからゾーンへの移動をするには、ある規程回数の戦闘をこなさなければなりません。
その回数をこなすまでは移動ボタンが出現しません。
レベル上げのために規程回数以上の戦闘をしてももちろんOKです。
1度戦闘を行うと次の戦闘ができるまでに規程時間待つ必要があります。
（これはカウントダウン表示されます）

各ゾーンによって出現する敵のレベル帯が違います。
いきなり遠出をすると痛い目にあうでしょう。
まずはゆっくりじっくりと自分にあったレベル帯のゾーンで経験を積んでください。

街のゾーン以外は「現在の冒険者」の名前が画面上部に表示されます。
その名前をクリックするとその人のステータス画面が開きます。

ゲーム中、右クリックをすることで、CHA,SVC,SVM,SVF等のステータスを確認する事が出来ます。
操作方法

　難しい操作は一切ありません。マウスとキーボードさえあれば大丈夫です。
ボタンクリックと文字入力だけです。
