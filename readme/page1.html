はじめに

推奨ブラウザ

  • インターネットエクスプローラー 5.0以上（以下、IE)
  • ネットスケープナビゲーター 6.0以上（以下、ネスケ）

IEに関しては特に問題ないと思われますが、
ネスケでプレーされる方に関しては「自キャラ画像」が表示されません。
ボタンを押すと閲覧できるようにはしてありますが
戦闘画面においては表示されないことに注意しておいてください。
（プレー出来なくはありません）
これは、使用しているタグがブラウザに対応していないためです。
著作権等

　このゲームにおける画像、スクリプト等のすべてに関しての著作権は
製作者であるスカイにあります。
　よって、無断転載、配布等を一切禁止といたします。
　また、プレーするにあたって、マナー等を守っていただくようお願いします。
リンク

リンクフリーです。
共有してくださるサイト様も募集中です。

重複登録

お一人様、１アカウントのみとします。
同一のＰＣから２つ以上のアカウントが発覚した場合、
予告なくアカウント削除及びＩＰブロックの措置を取らせて頂きます。
ご注意ください。

不正・荒らし行為

ブラウザの戻る・更新は使わないで下さい。
データの破損に繋がります。
ボタンの連打も禁止です。
サーバーに過大な負担を掛け、サーバーダウンに繋がります。
荒らし、嫌がらせ、又はそれに等しい行為。
上記の行為が発覚した場合（どれか１つでも当てはまった場合）、
アカウント削除及びＩＰブロックの対象となります。

その他

キャラのデータ保有期限は、２０日間です。
２０日間、１度も戦闘をしないと、データが削除されますのでご注意ください。
