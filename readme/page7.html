戦闘

バトルシステム

ターン制一括処理方式を採用しています。
簡単に言ってしまえば１度戦闘開始ボタンを押すと
あとは戦闘が終了するまで自動で処理してくれるということです。
１度プレーしてみればどんな感じかはすぐに判るはずです。

戦闘途中で回復アイテムの使用などのプレーヤー操作の介入は一切ありません。
よって戦闘前に自分のHPをしっかりチェックしておきましょう。
アイテム獲得

戦闘終了後にアイテムを落とすことがあります。
アイテム獲得率は以下のようになっています。
すべてランダムで決定されます。

    コモンほとんどがこのドロップに属します。一番確率が高い。雑魚敵は何もアイテムを持っていませんが、そのゾーンのボス級になると何か落とすかもしれません。
    レアコモンより出にくいドロップです。
    ウルトラレア１激レアなドロップです。優秀なアイテムが多い。一匹の敵で最高３種類まで持っていることがあります。
    ウルトラレア２
    ウルトラレア３

複数のゾーンに出現する同種の敵でも、
ゾーンによっては落とすアイテムが違う場合があります。
がんばってレアなアイテムをゲットして、みんなに自慢しましょう。

　　※アイテム欄がいっぱいだとアイテムは獲得できません。
ペット隊列

ペットを連れて歩く際に隊列というものが存在します。
その順番によって、戦闘参加のタイミングが変わってきます。

    １番目・・・プレーヤーより先に行動します。基本位置はここ。
    ２番目・・・プレーヤー行動後に行動します。
    ３番目・・・敵行動後に参加します。

複数のペットを連れて歩けるジョブは
このペット隊列の配置の仕方によって
大きく戦況が変わってくることでしょう。
キャラの死亡

HPがゼロになった時点でキャラ死亡です。
次のようなペナルティがあります。

  • スタートの街へ強制送還
  • 所持金が半額
  • 連れて歩いていたペットが逃げ出す
